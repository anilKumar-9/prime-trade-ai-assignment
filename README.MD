## Overview

**Prime Trade AI** is a role-based task management system developed as part of a technical assessment.  
The project is **backend-focused**, emphasizing authentication, authorization, and clean REST API design, with a minimal frontend for demonstration.

---

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- bcrypt

### Frontend
- React (Vite)
- React Router
- Axios

---

## Roles & Permissions

### Admin
- View all users
- Create and assign tasks
- View all tasks
- Delete any task

### User
- Register and login
- View only assigned tasks

---

## API Overview

### Authentication
- `POST /auth/register`
- `POST /auth/login`

### Users (Admin Only)
- `GET /users`
- `PUT /users/:id/promote`
